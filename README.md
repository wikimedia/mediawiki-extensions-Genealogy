# MediaWiki Genealogy extension

## Usage

Define this person's dates.
{{#genealogy:person |date of birth=Y-m-d }}

Define a parent:
{{#genealogy:parent | Page Name Here }}

List all siblings:
{{#genealogy:siblings}}

Define a partner:
{{#genealogy:partner | Page Name Here |start date=Y-m-d |end date=Y-m-d }}

List all partners:
{{#genealogy:partners}}

List all children:
{{#genealogy:children}}
